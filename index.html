تعلم أساسيات HTML و CSS و JavaScript بطريقة سهلة

HTML CSS JavaScript

HTML

HTML هي لغة تُستخدم لبناء هيكل صفحات الويب. تتكون من وسوم مثل العناوين والفقرات والصور والروابط.

<h1>مرحباً بالعالم</h1>
<p>هذه فقرة نصية.</p>

CSS

CSS هي لغة تُستخدم لتنسيق صفحات الويب، مثل الألوان والخطوط وتوزيع العناصر.

h1 {
  color: blue;
  text-align: center;
}

JavaScript

JavaScript تُستخدم لإضافة التفاعلية للموقع مثل الأزرار والأحداث.

function showMessage() {
  document.getElementById("output").innerText = "مرحباً! لقد ضغطت الزر.";
}
